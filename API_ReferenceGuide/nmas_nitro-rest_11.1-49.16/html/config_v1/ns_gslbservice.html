ns_gslbservice

Configuration for NS GSLB Services Information resource.

Read/write properties

svc_port<Integer>

Service Port.
Minimum value = 1
Maximum value =

hostname<String>

Hostname of the managed device.
Minimum length = 1
Maximum length = 256

ns_ip_address<String>

NS IP Address.
Minimum length = 1
Maximum length = 64

svc_ip_address<String>

Service IP Address.
Minimum length = 1
Maximum length = 64

svc_name<String>

Service Name.
Minimum length = 1
Maximum length = 100

id<String>

Id is system generated key.

save_config<Boolean>

Save configuration after gslb service enable/disable.

vsvr_name<String>

Vserver Name.
Minimum length = 1
Maximum length = 100

delay<Double>

Time, in seconds, allocated to the NetScaler appliance for a graceful shutdown of the service.

graceful<Boolean>

Graceful shutdown of Service..

Read only properties

svc_state<String>

Service State.

svc_type<String>

Service Type.

poll_time<Integer>

Last Polling Time.

display_name<String>

Display Name.

partition_name<String>

Partition Name.

svc_state_timestamp<Integer>

Service Last Statechange Time Stamp.

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE
