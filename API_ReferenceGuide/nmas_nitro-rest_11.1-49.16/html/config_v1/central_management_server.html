central_management_server

Configuration for Central managment server resource.

Read/write properties

password<String>

Password of central management server.
Minimum length = 1
Maximum length = 128

server<String>

Server/IP address of central management server.
Minimum length = 1
Maximum length = 64

type<String>

Type of central management server.
Minimum length = 1
Maximum length = 64

id<String>

Id is system generated key.

user_name<String>

User Name of central management server.
Minimum length = 1
Maximum length = 128

Read only properties
